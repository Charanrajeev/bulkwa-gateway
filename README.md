# BulkWA SMS Gateway

React Native Android app — SMS gateway for BulkWA Chrome Extension.

## How to Build APK (GitHub Actions — no PC needed)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** icon (top right) → **New repository**
2. Name it: `bulkwa-gateway`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload all these files
Click **uploading an existing file** on the repo page.
Upload ALL files from this zip, keeping the folder structure intact.

### Step 4 — Trigger the build
1. Click the **Actions** tab in your repository
2. Click **Build APK** in the left sidebar
3. Click **Run workflow** → **Run workflow**
4. Wait ~10-15 minutes for the build to complete ✅

### Step 5 — Download your APK
1. Click the completed workflow run
2. Scroll down to **Artifacts**
3. Click **BulkWA-Gateway-APK** to download
4. Unzip → install `app-debug.apk` on your Android phone

---

## Install on Phone
1. Transfer APK to phone (WhatsApp, Google Drive, USB)
2. Go to **Settings → Security → Install Unknown Apps**
3. Allow your file manager
4. Tap the APK → Install
